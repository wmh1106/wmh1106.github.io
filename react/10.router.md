# Router

## 路由的3个特点

历史：记录、以栈形式存储
跳转：从一个页面跳到另一个页面
事件：监听页面跳转时的事件

页面 Router

- location.href，每次进入页面都是重新加载
- history.back()

Hash  Router

- location.hash，不刷新，不重新加载
- window.onhashchange

H5 Router

- history.pushState('test','','#hash 或者 /user/index ')
- history.replaceState(name,title,path||hash)
- window.onpopstate

---

## 插件 React-Router

路由方式: `<BrowserRouter />` `<HashRouter />`

路由规则： `<Router />`

路由选项：`<Switch />` 多个匹配一个

跳转导航：`<Link />` `<NavLink />`

自动跳转：`<Redirect />`
