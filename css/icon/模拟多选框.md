# 模拟多选框（只有样式）

```html
<style>
    .checkbox{
        width: 16px;
        height: 16px;
        border:2px solid transparent;
        box-shadow: inset 0 1px,
                    inset 0 -1px,
                    inset 1px 0,
                    inset -1px 0;
        background-color: #fff;
        background-clip: content-box;
        color: #d0d0d0;
    }
</style>
<div class="checkbox"></div>
```
